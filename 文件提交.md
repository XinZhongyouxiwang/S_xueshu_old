
初次提交
- 初始化仓库
````
git init
````
- 提交缓存区
````
git add .
````
- 上传说明
````
git commit -m "上传说明"
````
- 推送缓存区
````
git remote add origin git@github.com:XinZhongyouxiwang/S_xueshu_old.git
````
- 提交
````
git push -u origin master
````
